How to install windows 11 on your pc: -

  1. Download iso file from the website
  2. Make bootable usb using rufus
  3. Insert the usb and reboot your device.
  4. Now press one time boot menu key on your pc and boot from usb.
  5. Now you may proceed with the steps as of we did for windows 10.

Highlights of Windows 11: -

  • Centralized start menu.
